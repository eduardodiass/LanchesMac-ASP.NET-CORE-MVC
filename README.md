-------
### 📍 Observações 📍

-------
 ### 🚀 Tecnologias 🚀
As tecnologias utilizadas foram:
* ✔️ ASP.NET Core MVC
* ✔️ SDK .NET 6
* ✔️ SQL Server
* ✔️ Entity Framework Core
* ✔️ ASP.NET Core Identity 
* ✔️ HTML, CSS, Javascript
* ✔️ Bootstrap 5
